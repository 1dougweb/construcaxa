# Sons de Notificação

Coloque seus arquivos de som MP3 nesta pasta.

Os arquivos devem ter extensão `.mp3` e serão listados automaticamente no seletor de som do dropdown de notificações.

Exemplo de nomes:
- notification1.mp3
- notification2.mp3
- alert.mp3
- bell.mp3

O sistema irá detectar automaticamente os arquivos MP3 nesta pasta.
